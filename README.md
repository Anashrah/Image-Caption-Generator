# Image-Caption-Generator
Repository for the computer vision project, Image Caption Generator.

An image captioning system that generates concise captions for input images. The project fine-tunes a pretrained model on example images and captions. An image is given as an input, and captions are generated as an output. It includes code for preparing data, training a model, and testing results.
